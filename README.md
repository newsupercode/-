#IDCarnumberCamera
车牌和行驶证拍照（横屏竖屏拍照 放大裁剪功能）

裁剪参考   

https://github.com/edmodo/cropper

相机拍照改动的

https://github.com/smartown/CertificateCamera

这个增加了调大调小焦距的功能  还有裁剪 压缩的功能 还解决了有的手机读取图片会旋转的问题
